   b = back | move uma palavra para trás
   e = end | move para o fim da palavra
   $ = eol | final da linha
   0 = bol | início da linha
   h =      
   j =      
   k =      
   l =      

   ( = vai para o começo do parágrafo
   ) = vai para o fim do parágrafo
